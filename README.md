# Project Title

Live Host Scanner 

## Description

Bash script to find live hosts on current network.

## Getting Started

### Executing program

* The .sh file should be given executable permission.
```
chmod +x /path/LiveHostScanner.sh
```

*Then it can be run via terminal.
```
/path/LiveHostScanner.sh
```
